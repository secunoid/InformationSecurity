# California Security Breach Information Act (S.B. 1386)
Tags: #securitybreach #PersonalData #personalinformation

----
## Overview
The California Security Breach Information Act defines covered “personal information” to include:
- social security number
- driver’s license number
- California Identification Card number
- Credit card number
- Debit card number
- Account number in combination with any required security code, access code, or password that would permit access to the account.
